# Sportradar task

Read the tests and run them to check how it works

### Installation

```bash
$ npm i
```

## Test

```bash
# unit tests
$ npm run test

# watch mode
$ npm run test:watch

# test coverage
$ npm run test:cov
```

## Code formatting

```bash
# lint with fix
$ npm run lint --fix

# prettier
$ npm run refactor
```
